# 📺 Lexi Watchlist

### Anime Watchlist, Planning & Discovery Tool

**Stop losing anime you'll never remember later. Build the watchlist you'll actually use.**

Save anime for later, organize thousands of titles, track your progress, and instantly find the perfect show when you're finally ready to watch something.

No more screenshots.

No more dozens of browser tabs.
